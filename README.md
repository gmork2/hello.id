# hello.id

Easily validate standardized codes and identifiers

This application is primarily designed to check whether the format of an identifier complies
with international standards and is therefore valid for use in that context.

Not only does this improve confidence in your data, but in case it can't be validated for the
type of data you expected, it also helps you determine what type of identifier it might be.
In the detail views you can get additional information about the applied validation rules
which can be interesting, especially for use in third-party applications like spreadsheets.

Valide fácilmente códigos e identificadores estandarizados

Esta aplicación está diseñada principalmente para verificar si el formato de un identificador
cumple con los estándares internacionales y, por lo tanto, es válido para su uso en ese contexto.

Esto no solo mejora la confianza en sus datos, sino que, en caso de que no se pueda validar para
el tipo de datos que esperaba, también lo ayuda a determinar qué tipo de identificador podría ser.
En las vistas de detalle puede obtener información adicional sobre las reglas de validación
aplicadas que puede ser interesante, especialmente para su uso en aplicaciones de terceros como
hojas de cálculo.
